# ARS.Dev — Portfolio

Personal portfolio website built with React + Vite + Tailwind CSS + Framer Motion.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
